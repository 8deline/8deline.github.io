Delivery pool

Delivery pool is a full-stack web application that allows users to search for people ordering food near them to save delivery cost.

Google geocoding API and mongodb $geonear operator are used to identify other people near you and sort based on distance from your location.

Technology used: JavaScript, HTML/CSS, RESTful APIs, Mongoose, Express, React, Node.js
